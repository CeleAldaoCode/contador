                                          CONTADOR- Proyecto simple pero funcional.

Un contador interactivo hecho con HTML, CSS, y JavaScript, ideal como ejercicio de lógica, manipulación de DOM y primeros pasos en proyectos web.
Permite sumar, restar y resetear el valor con botones dinámicos.

                                          Funcionalidades:

*Incrementar el contador.
*Decrementar el contador.
*Resetear a cero.
*Actualizacion en el tiempo real mediante JavaScript.
*Interfaz minimalista.

                                          Tecnologias Utilizadas:

*HTML
*CSS
*JavaScript

                                          Demo en vivo
                            https://celealdaocode.github.io/contador/

                                          Como usarlo:
                            No necesitás instalar nada.
                            Solo entra al sitio que tre deje arriba y usarlo directamente
